# 关于 
gin-demo基于gin使用相关的开源工具进行web项目开发。
本项目致力于帮助初学者掌握Go web开发，本项目保持简单！
本项目仅供学习参考！

主要使用以下第三方包：
- gin 项目路由
- gorm Go语言的Mysql关系型ORM
- go-redis Go语言的redis客户端
- viper 配置文件解析
- cors 网站请求API，跨域处理
- jwt-go 用户的身份校验
- zap 日志记录
- lumberjack 日志切割
